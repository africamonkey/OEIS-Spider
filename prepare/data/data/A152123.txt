    "comment": [
        "By definition, a commutative Moufang loop (CML) is a loop that satisfies the identities xy = yx, xy * zx = (xy * z)x. Let |L_n| = 3^delta(n). The Manin problem asks to calculate delta(n). The (hypothetic) Smith's formula gives the following values for delta(n) when n <= 7: delta(3) = 4, delta(4) = 12, delta(5) = 49, delta(6) = 220, delta(7) = 1007."
