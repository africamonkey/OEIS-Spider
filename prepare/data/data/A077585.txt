        "Let S be a set of n elements. First we perform a set partition on S. Let SU be the set of all nonempty subsets of S. As is well known, 2^n - 1 is the number of nonempty subsets of a set with n elements (see A000225). That is, 2^n - 1 is the number of elements |SU| of SU. In the second step, we select k elements from SU. We want to know how many different selections are possible. Let W be the resulting set of selections formed from SU. Then the number of elements |W| of W is |W| = sum((binomial(2^n-1,k)), k=1..2^n-1) = 2^(2^n-1)-1 = A077585.
